# 毕业设计论文笔记
## Single-Image Crowd Counting via Multi-Column Convolutional Neural Network
使用多列卷积神经网络，mae(平均绝对误差)，mse(均方误差)作为评价指标

使用卷积神经网络做人群计数主要有两种：
- 输入图片，直接输入图片中的人数
- 输入图片，转化为密度图，对密度图进行求和得到总人数

密度图 ： 密度图像包含信息更多，如人群的空间分布信息等，
同时也更适合视角各不相同的任意输入图像。

由于视角的差异，图像中的人头大小各不相同，因此固定大小的卷积核并不适合
用不同大小的卷积核来对应不同大小的人头。
